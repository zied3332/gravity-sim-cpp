# Gravity Simulation in C++

A simple first version of a gravity simulation built with C++ and Raylib.

## Features

- Multiple bodies attracted by gravity
- Real-time movement
- Pause and reset controls
- Simple visual simulation

## Controls

- SPACE: Pause / Resume
- R: Reset simulation
- ESC: Exit

## Run

```bash
mkdir build
cd build
cmake ..
cmake --build .
./GravitySimulation
